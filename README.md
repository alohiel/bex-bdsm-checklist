# Interactive Version of Bex Caputo's Yes/No/Maybe Worksheet

View it here: [alohiel.github.io/bex-bdsm-checklist](https://alohiel.github.io/bex-bdsm-checklist/)

This project is based on [Bex Caputo's](https://www.bextalkssex.com/) Yes/No/Maybe [worksheet](https://www.bextalkssex.com/wp-content/uploads/2016/10/BexCaputoYesNoMaybe.pdf).

## Available Scripts

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
